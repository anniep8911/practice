custom animation

.animate(properties, duration, easing, callback)

.animate(properties,option)

  1. properties

    객체형(속성과 값의 집합) 데이터를 사용하며, 속성과 속성값은 css에서 사용하는 속성과 값의 형태로 사용할 수 있으며 js에서 사용하는 객체속성과 값으로 사용할 수 있음

    사용예) {속성명:속성값, 속성명: 속성값, 속성명: 속성값}

  2. option

    객체형 데이터를 사용하며 정해져있는 속성과 속성값을 사용하여 애니메이션 효과를 도움

    • duration : num | str

      애니메이트가 지속되는 시간을 설정합니다.

    • easing: str

      jquery ui plug-in이 필요하며 등가속도 운동을통해 애니메이트되는 형태를 결정

    • complete:function

      현재 에니메이트가 완료되었을때 실행되는 함수

    • progress: function

      현재 애니메이트가 진행중일때 실행되는 함수
