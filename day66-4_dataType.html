여러 데이터 유형들

문자형 데이터: String

따옴표 안에 표기된 값을 문자형 데이터라고 합니다. 결합 연산자를 통해 문자와 문자를 결합할 수 있습니다.

숫자형 데이터 : Number

순수 숫자로만 표기된 값을 숫자형 데이터라고 합니다. 여러 연산자를 사용하여 연산된 결과를 얻을 수 있습니다. 경우에 따라 문자형 숫자도 사용하게 됩니다.

논리형 데이터: Boolean

연산에 대해서 true ,false로 결과가 나타나는것을 의미함 주로 비교연산자에서 변환되는 값을 이야기함

배열형데이터 : Array

한개의 값만 대입할 수 있는 변수를 여러개의 공간으로 나누는 데이터

객체형 데이터: Object

속성과 속성값을 나타내는 데이터로 CSS에서 사용하는 속성과 속성값 형태가 객체형 데이이터임.

함수형 데이터:Function

프로그래밍된 코드나 기능을 저장하는 데이터입니다. 형태에 따라 다양한 방식으로 기능을 구현할 수 있습니다.
