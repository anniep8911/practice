변수란?

Variable이란 ?: 변하는 데이터를 저장할 수 있는 메모리 공간입니다. 변수에는 오직 하나의 데이터만 저장이됩니다. 그러다보니, 새로운 데이터가 들어오면 기존에있던 데이터를 공간에서 지워버립니다.(덮어쓰기)

변수 선언

변수를 선언할때는 즉 변수를 만들깨는 var라는 키워드를 사용합니다. var키워드를 사용하고 난 이후에 저장소에 대한 이름을 만들어 주어야 합니다. 저장소의 이름은 원하는것으로 설정이 가능합니다.

변수 초기화

변수를 선언한 상태로 어떠한 값도 대인되지않은 상태에서는 null이 들어가있는 상태입니다. null값 대신 변수에 저장할 테이터 형태에 따라 기본값을 정의해 주는것이 좋습니다.

데이터 타입(자료형)

데이터: 변수에 대입 or 저장 되는 것들: 향태에따라서 분류

  1. string: "문자형"
  2. Number : 숫자형
  3. Boolean: 논리형(true/false)
  4. Array : 배열형
  5. Object : 객체형
  6. Function: 함수형
  7. Null: 빈값
