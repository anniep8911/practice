낮은수준의 시각효과

.hide() .show() .toggle()

.method([duration],[easing],[compleat])

  • [duration] : 효과의 지속시간

    효과가 적용되는 시간으로 숫자 형태의 밀리초(ms)단위나 문자형태의 키워드를 사용.
    1000 = 1s; 혹은 'fast' or 'slow'같이 문형태로 지속시간을 사용할 수 있음
    default는 0ms이며,매개변수가 여러개일 경우 400ms로 조정이 된다.

  • [easing] : 등가속도 운동을 나타냄

    jQuery core에는 easing의 함수가 내장되어있지않아 외부에서 ui plugin이 연결되어있는 상태여야 합니다.

  • [complete] : 콜백함수의 일종

    함수형태로 적용이되는 인자값입니다.
    효과가 실행이 완료되자마자 실행되는 함수.

basic effects

hieght속성과 opacity속성을 동시에 조절하여 보이거나 보이지 않게 하고 최종결과로 display속성을 none과 none이 아닌상태로 바꿔준다.

fading effects

fade는 opacity값만 가지고 전환을 시킴 display최종값은 none이나 none이 아닌상태로 변환을 시킴

sliding effects

height속성으로 요소가 보이거나 보이지 않게함 display 최종값은 none이거나 none이 아닌것으로 전환
