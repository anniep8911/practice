연산자의 종류

  1. 산술연산자

    • A+B : 플러스연산자
    • A-B : 마이너스 연산자
    • A*B : 멀티플 연산자
    • A/B : 디바이드 연산자
    • A%B : 모듈러스 연산자

  2. 대입연산자

    • A=B: 좌항에 우항을 대입
    • A+=B : A=A+B,좌항에 좌항과 우항을 더한값을 대입하는것
    • A-=B : A=A-B
    • A*=B : A=A*B
    • A/=B : A= A/B

  3. 비교연산자

    • A==B : 좌항과 우항은 같다
    • A!=B : 좌항과 우항은 같지않다 (다르다)
    • A>B: 좌항이 우항보다 크다(greater than)
    • A<B: 좌항이 우항보다 작다(less than)
    • A===B : 좌항과 우항은 완전같다(데이터 유형까지 같은 경우를 이야기 함)

  4. 증감연산자

    값을 1씩 증가하거나 감소하는 연산입니다.

    • ++a : a에 1을 증가시킴(대입 전) :A=A+1
    • a++ : a에 1을 증가시킴(대입 후)
    • --a : a에 1을 감소시킴(대입 전): A=A-1
    • a-- : a에 1을 감소시킴(대입 후)
